## Objetivo

Desenvolver um sistema de gestão de teatro com backend, aplicação desktop e estrutura preparada para módulo web, permitindo apoiar a administração de espetáculos, sessões, salas, lugares, preçários e relatórios.

## Tech Stack

- Java 21
- Spring Boot
- Spring Security
- JavaFX
- Maven
- PostgreSQL
- Docker Compose

## Estrutura do Projeto

- `backend/` - API REST e lógica de negócio
- `desktop/` - aplicação desktop para administração
- `database-init/` - scripts de inicialização da base de dados
- `web/` - futuro módulo web
- `docker-compose.yml` - configuração do PostgreSQL
